
## Vim config

After stowing, should be able to just run `:PlugInstall` as the only additional step to install plugins.

The leader character is a comma `,`

Auto format is set to `<Leader-F>` or `,f`

## Fuzzy Search

> For searching files in root directory

Mapped to <CTRL-P>

## NerdTree

> For navigating filesystem

Mapped to <CTRL-T>

## BufferList

> For switching between buffers

Mapped to `<Leader>-b`, so `,b`

## MultiCursor

> For editing multiple instances

Mapped to `<Alt>-N` for all and `<CTRL-N>/<CTRL-P>`  for next/prev

## Goyo / Limelight

> For distraction free editing

Mapped to `<Leader-G>`, so `,g`

## YouCompleteMe

> For intellisense autocomplete

You will have to run the installation script in `~/.vim/plugged/YouCompleteMe`
